# PodcastHub — Smart Podcast Recommendation System

A full-stack web application that helps users discover podcasts 
tailored to their interests using a hybrid recommendation engine.

## Tech Stack
- **Frontend:** React.js, HTML, CSS
- **Backend:** Node.js, Express.js
- **Database:** MongoDB
- **Authentication:** JWT (JSON Web Tokens)
- **API:** RESTful APIs

## Features
- Hybrid recommendation engine (content-based + collaborative filtering)
- Role-based dashboards — Listener, Creator, and Admin
- Podcast streaming and content management
- Personalised recommendations based on user behaviour
- Analytics and engagement tracking
